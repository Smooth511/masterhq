--- WHAT DD CHANGES ---

dd = raw disk imaging. You can now:

  # Image sda BEFORE wiping (preserve whatever's on it):
  dd if=/dev/sda of=/mnt/s1/sda-original.img bs=4M status=progress

  # Wipe sda before formatting (ensures clean state):
  dd if=/dev/zero of=/dev/sda bs=4M status=progress count=10
  # (just wipe the first 40MB = partition table + fs headers, that's enough)

  # After full takeover: image the trap disk (evidence preservation):
  dd if=/dev/sda of=/mnt/s1/trap-evidence.img bs=4M status=progress

--- WHAT CPIO CHANGES ---

cpio replaces cp -a and rsync for the COW delta copy. More reliable on deep trees:

  cd /start/mid/upper/cow/upper && find . | cpio -pdm /mnt/s1/lower/

The -p flag = pass-through (copy to directory), -d = create dirs, -m = preserve mtimes.
This is the standard initramfs copy method.

=== PHASE 4: BUILD THE OVERLAY ===

  -o lowerdir=/mnt/s1/lower,upperdir=/mnt/s2/upper,workdir=/mnt/s2/work \
  /mnt/owned && echo OVERLAY_OK

ls /mnt/owned/etc/ | head -5


=== PHASE 5: CHROOT IN ===

mount -t proc proc /mnt/owned/proc

mount --bind /dev /mnt/owned/dev

chroot /mnt/owned /bin/bash
