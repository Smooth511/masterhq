# Malware Database

A collection of markdown documents, logs, investigations, and information related to malware analysis and research.

## Structure

| Directory | Contents |
|-----------|----------|
| `malware/` | Profiles and technical details on specific malware families |
| `investigations/` | Case-based investigation documents and write-ups |
| `logs/` | Raw and annotated log files from malware analysis |
| `reports/` | Analysis reports, summaries, and findings |
| `history/` | Chronological timeline of events, battles, and milestones |

## Usage

Browse by category using the directories above.  Each entry follows the template
provided in its directory.  Fork this repository to maintain your own copy of the
database.

## Contributing

1. Pick the appropriate directory for your data.
2. Copy the relevant template file and fill it in.
3. Submit a pull request.
